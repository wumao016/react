##### css是项目样式文件，包含所有的css文件以及less文件
##### data是异步接口数据文件夹，包含所有的异步接口数据
##### fonts是字体文件夹，由bootstrap提供的字体
##### img是图片文件夹，包含所有的图片
##### js是脚本文件，包含库文件以及组件文件
##### index.html 项目入口文件
##### fis-conf.js  工程化配置文件
